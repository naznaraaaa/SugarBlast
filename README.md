# SugarBlast

SugarBlast adalah game puzzle Match-3 berbasis Unity yang mengharuskan pemain mencocokkan minimal tiga candy yang sama untuk memperoleh skor.

## Fitur
- 3 Level permainan
- Sistem skor
- High Score
- Level Unlock
- Timer
- Pause Menu

## Teknologi
- Unity 2D
- C#
- TextMeshPro
- PlayerPrefs
